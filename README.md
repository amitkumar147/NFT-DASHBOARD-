# Rare 3D Holographic NFT Dashboard + AI Assistant

## Features
- Wallet connect (simulated)
- Mint NFT → 3D rotating holographic cube
- Animated neon particle background
- Typing-style AI assistant notifications
- Dark/Light mode toggle
- Fully mobile-friendly (Acode demo)
- GitHub-ready structure

## How to Run
1. Open `index.html` in browser or Acode
2. Click "Connect Wallet"
3. Click "Mint NFT" → see holographic NFT cubes with AI-style notifications and particle animation
